5. Corrector Ortográfico: atributo spellcheck

spellcheck no funciona (todavía) en todos los navegadores. Se puede cambiar idioma con lang="en".


lang="en" spellcheck="true"

7. Modales con <dialog>

Esto es una etiqueta dialog cerrada

Botones apra abrir y cerrar dialog

Texto dentro del dialog que ahora puedes cerrar clicando X. 


<button onclick="miDialog.showModal()">Abrir modal</button>
<dialog id="miDialog">
    <button id="X" 
    style="display:block; margin-right: 0; margin-left: auto;"
    onclick="miDialog.close()"
    >
    Cerrar [X]
    </button>
    <p>
        Texto dentro del dialog que ahora puedes cerrar clicando X.
    </p>
</dialog>

8. <label> con <input>

Radio buttons
Checkboxes 

<form action="#">
    <div>
        <label for="text-input" style="border: 1px solid brown; width: 100%; display:block">Etiqueta del input (haz clic)</label>
        <input id="text-input" style="display: block; width: 100%; margin-top: 5px;" type="text" placeholder="Texto...">
    </div>
    
    <fieldset id="radio-btn">
        <legend>Radio buttons</legend>
        <label for="radio1">Radio button</label>
        <input id="radio1" name="radio-btn" type="radio">
        <label for="radio2">Radio button con más texto</label>
        <input id="radio2" name="radio-btn" type="radio">
        <label for="radio3">Radio button con poco texto</label>
        <input id="radio3" name="radio-btn" type="radio">
    </fieldset>

    <div>
        <fieldset id="checkboxes">
            <legend>Checkboxes</legend>
            <label for="checkbox1">Checkbox</label>
            <input id="checkbox1" type="checkbox">   
            <label for="checkbox2">Checkbox</label>
            <input id="checkbox2" type="checkbox"> 
            <label for="checkbox3">Checkbox</label>
            <input id="checkbox3" type="checkbox">                  
        </fieldset>
    </div>
</form>
